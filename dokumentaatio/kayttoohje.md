# Käyttöohje

## Ohjelman käynnistäminen

- Asenna riippuvuudet komennolla: 

````          poetry install````

- Luo database ennen sovelluksen käynnistämistä:

````  poetry run invoke build````

- Käynnistä sovellus komennolla:

````  poetry run invoke start````
